# Projeto Ecommerce-ebook

## Desevolvido por: 
    @maria-baeta

## Objetivo
 
  Desenvolver uma aplicação que simule um Ecommerce de ebook que permita: 

      Usuário:  
        Se cadastra no sistema;
        Logar;
        Alterar seus dados;
        Selecionar os livros que deseje compra;
        Efeturar a compra(pagar);
        Deslogar

      Gerenciador do sistema: 
        Cadastrar um livro;
        Editar o livro;
        Excluir o livro;

É necessário: 
    
  ### Back-end:
